üìù User Registration Flow

  • User fills registration form (Name, Email, Password, Phone)
  • System validates email format and checks for duplicates
  • Password is hashed using bcryptjs (10 salt rounds)
  • Verification token is generated using crypto
  • User record created with isVerified: false
  • Verification email sent with unique link
  • User clicks link ‚Üí Account verified
  • JWT token issued ‚Üí Redirect to dashboard
